## 🚀 About Me

🎓 I'm a *Full Stack Developer* passionate about building impactful AI and cloud-based solutions.  
💡 I enjoy converting ideas into production-ready apps with a modern UI and smart backend.  
🛠️ Creator of *Omni Summarizer, an AI-powered tool that summarizes **text, PDFs, images, and videos, with **real-time interaction and dark mode*.

## 💼 Worked On

Here are some of the real-world projects and technologies I’ve worked with:

- 🔍 *Omni Summarizer* – All-in-one summarization tool (Text, PDF, Image, Video) using T5, BLIP, ViT-GPT4, Gemini  
- 🎥 *Video Summarizer* – Text summary + keyframes + preview video playback  
- 🖼️ *Image Summarizer* – Auto-captioning, Q&A, storytelling with multiple images  
- 📄 *PDF & Text Summarizer* – Compare outputs from different AI models  
- 📊 *Admin Dashboard* – Responsive, tracks user activity and summarizer usage
